## 승인 요청 프로젝트

### 1. APEX ML Platform
**공개 설명안**:
> "Distributed AutoML platform with GPU optimization and automated ML pipeline for prescriptive analytics"
> (분산 AutoML 플랫폼: GPU 최적화 및 처방적 분석을 위한 자동화 ML 파이프라인)

**다이어그램 내용**:
- 고수준 3단계 아키텍처 (Explore → Optimize → Evolve)
- 데이터 흐름만 표시
- **제외 사항**: 알고리즘 상세, 성능 지표, 구현 세부사항, 고유 기술

**용도**: 학술 포트폴리오에서 현재 업무 소개용

---

### 2. Acoustic Vision System
**공개 설명안**:
> "AI-powered sound visualization system using sensor fusion of microphone arrays and vision for anomaly detection"
> (AI 기반 음향 시각화 시스템: 마이크 어레이와 비전 센서 융합을 통한 이상 탐지)

**다이어그램 내용**:
- 개념적 시각화: ROI 기반 빔포밍 접근법 표시
- 센서 융합 개념도
- **제외 사항**: 구현 세부사항, 성능 데이터, 고유 알고리즘

**용도**: 학술 포트폴리오에서 현재 연구 영역 소개용
