Overview

I designed and built VisionForge as a solo developer handling both backend and frontend end-to-end, with guidance from my team manager. This production-grade AI vision platform serves PCB (Printed Circuit Board) defect inspection in manufacturing environments. I implemented IPC-A-610 compliant defect categories through a multi-model architecture combining YOLOv8 for detection, Stable Diffusion for synthetic data augmentation, and NeRF for 3D reconstruction.

Note: This project was developed for industrial clients under NDA. Specific client names and proprietary datasets have been anonymized. The methodology and architecture described here accurately represent my contributions.

My Contributions

  • System Architecture: I designed the complete end-to-end PCB inspection pipeline from camera input to IPC-A-610 compliance classification
  • Object Detection: I trained and optimized YOLOv8 models for multi-class defect detection (solder bridges, insufficient solder, cold joints, component misalignment)
  • Data Augmentation: I implemented Stable Diffusion-based synthetic defect generation to address class imbalance in training data
  • 3D Reconstruction: I integrated NeRF and Open3D for 3D solder joint inspection and quality analysis
  • Backend & Frontend: I built the complete FastAPI backend and React-based live inspection interface for production line integration

System Architecture


I designed this end-to-end PCB inspection pipeline from high-resolution camera input through YOLOv8 detection engine to IPC-A-610 compliance classification. I incorporated Stable Diffusion for synthetic defect augmentation, integrated NeRF+Open3D for 3D reconstruction, and built the FastAPI backend for seamless production line integration.

Defect Detection Dashboard


The main dashboard provides a comprehensive view of PCB inspection results, displaying detected defects with bounding boxes, confidence scores, and defect classifications in real-time. The interface supports batch processing and individual board analysis with IPC-A-610 compliance checking.

Macro Closeup Analysis


High-resolution macro closeup view enables detailed inspection of individual components and solder joints, supporting precise defect identification at the microscopic level. The system analyzes solder bridges, insufficient solder, cold joints, and component placement errors.

Live Inspection Interface


The live inspection interface provides real-time feedback during production, with instant defect detection overlays and pass/fail status indicators for seamless quality control integration. GPU acceleration enables processing speeds suitable for high-throughput production lines.

Multi-Board Batch Processing


Batch processing mode enables efficient inspection of multiple PCBs simultaneously, with aggregated statistics and defect distribution analysis across production batches. Quality metrics and trend analysis help identify systematic production issues.

Model Training Evolution


Visualization of the model training process showing loss curves, accuracy metrics, and performance evolution over training epochs. The system uses Stable Diffusion for synthetic data augmentation and NeRF for 3D reconstruction to enhance training data quality.
